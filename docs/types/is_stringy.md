## Description

`is_stringy()` - can we use the variable as a string?

```php
boolean is_stringy(mixed $item);
```

`is_stringy()` is a convenience wrapper around `IsStringy::check()`. See [`IsStringy::check()`](IsStringy.check.html) for details.
