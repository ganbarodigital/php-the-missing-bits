Property Helpers

Introduction

We've added some helper functions for inspecting and understanding class and object properties.

Available Functions

Function Purpose
FilterProperties::from() return a filtered list of a class's or object's properties
HasFilteredProperties::check() does a class or object have properties that pass a user-supplied filter?
