Combine Arrays

Introduction

We've added new ways to combine arrays. These are alternatives to PHP's built-in array_merge(). In real-life use, these new functions are sometimes faster.

Available Functions

Function Purpose
array_append_values() append values to an array, ignoring array keys
array_merge_keys() combine two arrays
